Most people even use the terms “Machine Learning”, “Deep Learning” and “Artificial Intelligence” interchangeably, however, each of these interlinked terminologies are unique and useful in its own way.

Artificial intelligence is the leveraging of computers and machines to mimic the problem-solving and decision-making abilities of a human mind. 🧠

Machine Learning uses interconnected nodes (like neurons) in a layered structure that resembles the human brain. It creates an adaptive system that computers use to learn from their mistakes and improve continuously.The structure/model used to do this is a Neural Network. It mimics the human brain through a set of algorithms.

In its simplest form, a Neural Network only has three layers:
1. The input layer
2. Hidden layer
3. The output layer

A neural network with multiple hidden layers with multiple nodes is known as a deep learning system, or a deep neural network (which helps predicts an output from complex data).so to recap: A Neural Network is the structure of Machine Learning for >Artificial Intelligence
